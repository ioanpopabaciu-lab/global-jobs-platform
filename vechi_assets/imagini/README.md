# Imagini de pe Site-ul Vechi GJC.ro

Imaginile nu au putut fi descărcate automat din cauza protecției anti-bot (BotGuard) activată pe server.

## Instrucțiuni pentru Descărcare Manuală:

1. Deschide un browser (Chrome, Firefox, etc.)
2. Navighează la fiecare URL de mai jos
3. Click dreapta pe imagine -> "Save Image As..."
4. Salvează în acest folder
